卡券类型：
- CASH代金券
- DISCOUNT折扣券
- GIFT兑换券
- GENERAL_COUPON优惠券
- MEMBER_CARD会员卡

#### 上传商户Logo
需要 `access_token`
请求方式：`POST`
<!-- more -->
请求地址：
```php
//微信API
config('wechat.api_uri') = 'https://api.weixin.qq.com';
/**
 * 上传图片
 */
const API_UPLOAD_IMAGE = '/cgi-bin/media/uploadimg';
```
请求参数：文件数据流buffer

#### 创建卡券
需要 `access_token`
请求方式：`POST`
请求地址：
```php
/**
 * 创建卡券
 */
const API_ADD_CARD = '/card/create';
```

```php
$data = [
    'card' => [

        'member_card' => [

                    "type" => "DATE_TYPE_FIX_TIME_RANGE",
```
